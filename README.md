
# ❤️ Дневник давления — Telegram-бот

Полностью бесплатный бот для ведения дневника артериального давления.

## Что умеет

- 📸 Распознаёт фото тонометра (Tesseract OCR — бесплатно)
- ✏️ Ручной ввод: пишете `130/85 72` и готово
- 💊 Записывает лекарства и заметки о самочувствии
- 📋 Показывает историю и статистику за неделю
- 🖨 Экспортирует таблицу для печати

---

## Настройка: пошаговая инструкция для новичка

### Шаг 1. Создайте бота в Telegram (2 минуты)

1. Откройте Telegram, найдите **@BotFather**
2. Отправьте: `/newbot`
3. Придумайте имя: `Мой дневник давления`
4. Придумайте username: `my_bp_diary_bot` (должен кончаться на `bot`)
5. BotFather пришлёт **токен** — строка вида `7123456789:AAHx...`
6. **Скопируйте токен** — он нужен на шаге 3

---

### Шаг 2. Загрузите код на GitHub (3 минуты)

1. Зайдите на **https://github.com**, зарегистрируйтесь если нужно
2. Нажмите **+** → **New repository**
3. Название: `bp-diary-bot`, выберите **Private**
4. Нажмите **Create repository**
5. На странице нового репозитория нажмите **uploading an existing file**
6. Перетащите **все файлы** из этой папки:
   - `bot.py`
   - `requirements.txt`
   - `Dockerfile`
   - `.gitignore`
7. Нажмите **Commit changes**

---

### Шаг 3. Запустите на Railway (5 минут)

Railway — сервис для запуска приложений. Есть бесплатный триал, потом $5/мес.

1. Зайдите на **https://railway.app**
2. Нажмите **Login** → войдите через GitHub
3. Нажмите **New Project** → **Deploy from GitHub repo**
4. Выберите `bp-diary-bot`
5. Railway начнёт сборку (первый раз ~2 минуты)

#### Добавьте переменную окружения:

1. Кликните на ваш сервис в Railway
2. Вкладка **Variables**
3. Нажмите **+ New Variable**
4. Введите:
   - **Name:** `TELEGRAM_TOKEN`
   - **Value:** ваш токен от BotFather
5. Railway перезапустит бот автоматически

**Готово!** Через 30 секунд бот заработает.

---

### Шаг 4. Проверьте

1. Откройте вашего бота в Telegram
2. Нажмите **Start** или отправьте `/start`
3. Попробуйте ввести: `120/80 72`

## Как пользоваться

### Добавить запись

**Фото:** сфотографируйте экран тонометра → отправьте → бот покажет распознанные числа → подтвердите или исправьте вручную.

**Текст:** напишите `130/85 72` (давление и пульс) или просто `130/85`.

Бот потом спросит про лекарства и самочувствие (можно пропустить).

### Команды

| Команда | Что делает |
|---------|-----------|
| `/start` | Приветствие |
| `/history` | Последние 10 записей |
| `/history_all` | Все записи |
| `/stats` | Статистика за неделю + тренд |
| `/export` | HTML-файл для печати |
| `/delete` | Удалить последнюю запись |
| `/cancel` | Отменить текущий ввод |
| `/help` | Справка |

### Распечатать

1. Отправьте `/export`
2. Откройте полученный файл в браузере
3. Ctrl+P (Cmd+P на Mac) → Печать или «Сохранить как PDF»

## Про распознавание фото

Бот использует Tesseract OCR — бесплатный движок распознавания текста.

**Советы для лучшего распознавания:**
- Фотографируйте **только экран** тонометра крупным планом
- Экран должен быть **хорошо освещён**
- Фото должно быть **в фокусе**, не размытое
- Бот покажет распознанные числа и спросит подтверждение

Если распознавание не сработало — просто введите цифры вручную.

## Решение проблем

**Бот не отвечает:**
- Проверьте в Railway: статус должен быть «Active»
- Убедитесь, что `TELEGRAM_TOKEN` правильный (без пробелов)
- Посмотрите логи: в Railway → Deployments → выберите деплой → View Logs

**Фото не распознаётся:**
- Сделайте фото крупнее и чётче
- Введите вручную: `120/80 72`

**Стоимость:**
- Telegram бот — бесплатно
- Tesseract OCR — бесплатно
- Railway — бесплатный триал, затем от ~$5/мес
- Альтернатива: бесплатный Render.com (выберите Docker при деплое)
